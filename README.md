# 宝可梦游戏攻略文档 - 使用说明

## 📖 项目简介

这是一个为游戏微信群创建的本地HTML文档系统，用于展示宝可梦游戏的各种攻略、数据和配方信息。无需部署服务器，直接在浏览器中打开即可访问。

## 🚀 快速开始

### 方法一：直接打开（推荐）

1. 找到项目文件夹中的 `index.html` 文件
2. 双击文件，用浏览器打开
3. 开始浏览游戏攻略文档

### 方法二：使用本地服务器

如果遇到跨域问题或需要更好的体验，可以使用本地服务器：

**使用 Python:**
```bash
python -m http.server 8000
```

**使用 Node.js (http-server):**
```bash
npx http-server -p 8000
```

然后在浏览器中访问: `http://localhost:8000`

## 🎯 功能说明

### 1. 主页 (index.html)
- 网站导航
- 搜索功能
- 快速访问各个板块

### 2. 游戏数据 (game-data.html)
- **宝可梦图鉴**: 查看各种宝可梦的详细信息
- **道具列表**: 游戏中的各种道具
- **属性筛选**: 按属性筛选宝可梦
- **排序功能**: 按名称或类型排序

### 3. 栖息地配方 (habitat-recipes.html)
- **基础栖息地**: 森林、海洋、火山、城市等
- **高级栖息地**: 冰雪、雷电、幽灵、花园等
- **配方详情**: 所需物品、产出宝可梦、摆放建议
- **建设技巧**: 提高产出的方法和注意事项

### 4. 图片展示 (gallery.html)
- **游戏截图**: 展示游戏中的各种场景
- **地图展示**: 世界地图和各个区域地图
- **角色立绘**: 宝可梦角色图片
- **道具图片**: 游戏道具图片

## 🎨 自定义内容

### 修改游戏数据

编辑 `game-data.html` 文件，找到相应的数据卡片进行修改：

```html
<div class="data-card" data-type="电">
    <h3>⚡ 皮卡丘</h3>
    <p><strong>类型:</strong> 电</p>
    <p><strong>描述:</strong> 著名的电属性宝可梦，脸颊两侧有电力袋</p>
    <p><strong>特性:</strong> 静电</p>
</div>
```

### 添加栖息地配方

编辑 `habitat-recipes.html` 文件，添加新的配方卡片：

```html
<div class="recipe-card">
    <h3>🏠 新栖息地</h3>
    <p><strong>描述:</strong> 栖息地描述</p>
    <div class="recipe-items">
        <span class="recipe-item">物品1 x数量</span>
        <span class="recipe-item">物品2 x数量</span>
    </div>
    <div class="recipe-output">
        <strong>🎁 产出宝可梦:</strong> 宝可梦1、宝可梦2
    </div>
</div>
```

### 添加图片

编辑 `gallery.html` 文件，替换占位符图片：

```html
<div class="gallery-item" onclick="openLightbox('图片路径', '图片描述')">
    <img src="图片路径" alt="图片描述">
    <div class="gallery-caption">图片描述</div>
</div>
```

### 更新搜索数据

编辑 `script.js` 文件中的 `gameData` 对象：

```javascript
const gameData = {
    pokemon: [
        { id: 1, name: "皮卡丘", type: "电", description: "描述" }
    ],
    // ... 其他数据
};
```
